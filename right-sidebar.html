Electronics projects

Worked on IoT based project "Underwater ROV" involving Arduino Concept and FM transmission and reception.

The project involved an Arduino Pro Mini board and RF module.This RF module consists of an RF Transmitter and an RF Receive operating at a frequency of 434 MHz. The L293D is a Motor Driver IC, it allows the motor to drive on both directions.By the use of rotor blades, the ROV can be propelled in water.

Java Project - Banking

Built a Java based project for automating Interest Calculation by finding Principal, Rate of Interest, Maturity Amount,Recurring Deposit and Savings by basic bank regulations and calculations.

Java Project - Credit Card Bill Generator

Built a Java based project for automating Credit card bill generation by finding Monthly transactions, Outstanding amount, Credit card number, Date of transaction, Amount transacted and Transaction type by basic banking credit regulations.
